### 1.2.4 Wounds

ICD10 CODE: S00–T88

Any break in the continuity of the skin or mucosa or disruption in the integrity of tissue due to injury.

#### Causes

- Sharp objects, e.g. knife, causing cuts, punctures
- Blunt objects causing bruises, abrasions, lacerations
- Infections, e.g. abscess
- Bites, e.g. insect, animal, human
- Missile and blast injury, e.g. gunshot, mines, explosives, landmines
- Crush injury, e.g. RTA, building collapse

#### Clinical features

- Raw area of broken skin or mucous membrane
- Pain, swelling, bleeding, discharge
- Reduced use of affected part
- Cuts: sharp edges
- Lacerations: irregular edges
- Abrasions: loss of surface skin
- Bruises: subcutaneous bleeding e.g. black eye

---

#### Management

| TREATMENT | LOC |
|---|---|
| **Minor cuts and bruises** | HC2 |
| □ First aid, tetanus prophylaxis, dressing and pain management | |
| □ Antibiotics are not usually required but if the wound is grossly contaminated, give: | |
| – Cloxacillin or amoxicillin 500 mg every 6 hours as empiric treatment | |
| – Child: 125–250 mg every 6 hours | |

| TREATMENT | LOC |
|---|---|
| **Deep and/or extensive wounds** | HC4 |
| □ Identify the cause of the wound or injury if possible | |
| □ Wash affected part and wound with plenty of water or saline solution | |
| □ Clean with chlorhexidine 0.05% | |
| □ Hydrogen peroxide 6% diluted with equal amount of saline to 3% if wound is contaminated | |
| □ Explore the wound under local anaesthesia to ascertain extent of the damage and remove foreign bodies | |
| □ Surgical toilet: carry out debridement to freshen the wound | |
| □ Tetanus prophylaxis, pain management, immobilization | |

---

#### Wound closure

| TREATMENT | LOC |
|---|---|
| **If wound is clean and fresh (<8 hours)** | HC3 |
| □ Carry out primary closure by suturing under local anaesthetic | |
| – Use lignocaine hydrochloride 2% (dilute to 1% with equal volume of water for injection) | |

| TREATMENT | LOC |
|---|---|
| **If wound is >8 hours old or dirty** | |
| □ Clean thoroughly and dress daily | |
| □ Check the state of the wound for 2–3 days | |
| □ Carry out delayed primary closure if clean | |
| – Use this for wounds up to 2–4 days old | |

| TREATMENT | LOC |
|---|---|
| **If wound >4 days old or deep puncture wound, contaminated wounds, bite/gunshot wounds, abscess cavity** | |
| □ Let it heal by secondary closure (granulation tissue) | |
| □ Dress daily if contaminated/dirty, every other day if clean | |
| □ Pack cavities (e.g. abscesses) with saline-soaked gauzes | |

| TREATMENT | LOC |
|---|---|
| **In case of extensive/deep wound** | |
| □ Consider closure with skin graft/flap | |

---

!!! note
    - Use SOP for collection of wound discharge, or deep tissue, submit to lab.  
    - Start treatment and change treatment when results return.  
    - If MDR, gram-negative organisms, MRSA or VRE are identified, implement respective transmission-based precautions.  
    - Where possible, use chlorine release for environmental decontamination or alternative fumigation (not formaldehyde).
