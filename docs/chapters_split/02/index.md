# Chapter 2: Infectious Diseases

This chapter covers infectious diseases commonly encountered in Uganda, including bacterial, fungal, viral, helminthic, and protozoal infections. It includes guidance on anthrax, brucellosis, diphtheria, leprosy, meningitis, plague, septicaemia, tetanus, typhoid fever, viral haemorrhagic fevers, COVID-19, intestinal worms, schistosomiasis, malaria, and human African trypanosomiasis.

Use this chapter to support diagnosis, treatment, prevention, infection control, and referral decisions for infectious diseases across levels of care.
