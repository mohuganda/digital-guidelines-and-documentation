# Chapter 9: Mental, Neurological and Substance Use Disorders

This chapter covers neurological disorders, psychiatric conditions, and substance use disorders. It includes epilepsy, nodding disease, headache, migraine, dementia, parkinsonism, delirium, anxiety, depression, postnatal depression, bipolar disorder, psychosis, alcohol use disorders, substance abuse, childhood behavioural disorders, and childhood developmental disorders.

Use this chapter to support recognition, initial management, counselling, follow-up, and referral for mental, neurological, and substance use conditions.
