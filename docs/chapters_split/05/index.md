# Chapter 5: Respiratory Diseases

This chapter covers non-infectious and infectious respiratory diseases, including asthma, chronic obstructive pulmonary disease, bronchiolitis, bronchitis, common cold, epiglottitis, influenza, laryngitis, croup, pertussis, pneumonia, lung abscess, and tuberculosis.

Use this chapter to support respiratory assessment, diagnosis, treatment, infection control, prevention counselling, and referral across age groups.
